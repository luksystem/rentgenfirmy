# Ikony PWA / Web Push

Dodaj następujące pliki PNG (na podstawie logo firmy z `/ustawienia/firma`):

| Plik | Rozmiar | Użycie |
|------|---------|--------|
| `push-icon-192.png` | 192×192 | Manifest PWA, ikona powiadomienia |
| `push-icon-512.png` | 512×512 | Manifest PWA (install prompt) |
| `push-badge-96.png` | 96×96 | Badge powiadomienia (monochromatyczna) |

Bez tych plików push i PWA działają, ale przeglądarka może pokazać domyślną ikonę.
